Every antipattern belongs to one of 12 categories. Categories are the primary navigation dimension of the library — you can filter by them on the `/library` page or browse each one directly at `/category/[slug]`.

<Tip>
  The homepage renders the full 12-category grid in the **Pattern Categories** section. Click any card to go directly to that category's collection.
</Tip>

## Category URLs

Each category is accessible at `/category/[slug]` in the Bad UI application. The `slug` column in the table above is the URL segment. For example, the Accessibility category is at `/category/accessibility`.

The library filter panel on `/library` generates category filter buttons dynamically from published patterns — only categories with at least one live entry appear as a button.

## Filtering the library by category

You can scope the `/library` archive to a single category two ways:

1. Click a category button in the **Focus Area** filter panel on `/library`.
2. Navigate directly to `/category/[slug]` to view the full collection for that category.

<Note>
  Category filter buttons on `/library` are built from the live Supabase data, so they reflect only categories that have at least one published pattern.
</Note>
